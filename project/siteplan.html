Site Home

- Site Name -

Tic-Tac-Click

(tic-tac-click.com)

The reason for this name is that it is clearly connected to Tic-tac-toe, while also including a 'click', like the click of a mouse. The domain is available at the time of writing.

- Site Purpose -

This site provides a place to play a simple game of Tic-tac-toe, and allows the user to track winners using LocalStorage.

It also introduces users to basic Tic-tac-toe rules and more advances strategies.

- Scenarios -

Possible question site visitors might ask:

  • How do you play Tic-tac-toe?
  • What can I do to win more often?
  • How can I make the game more exciting?
